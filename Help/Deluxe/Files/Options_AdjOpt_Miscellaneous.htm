Adjust Options - Miscellaneous

Other adjustable options may be found by clicking the "Miscellaneous" button (after first clicking "Adjust Options").

Scroll Direction

Picture Scroller 2.1 will scroll through your selected pictures in the direction that you choose.
"Forward" - Pictures will be displayed from the first on the list to the last.
"Backward" - Pictures will be displayed from the last to the first.
"Random" - Pictures will be displayed in random order.

NOTE: No one picture is ever repeated until all other pictures have been shown.

Picture Size

This sets how large the picture will be displayed on the screen.  The meaning of each setting is described below:

"Original"

 		The picture will be shown on the screen in its original size.  Even if the picture is too large to fit on the screen, it will be shown in the center of the screen.
"Proportional" The picture is adjusted to fit the screen perfectly without distorting it.  (DEFAULT)
"Screen Size" The picture will fit the screen exactly, even if it must be distorted to do so.
"Tiled" The picture in its original size will be tiled across the screen.

Background Music

The "Background Music" setting has three options:

  1. No background music
  2. Background music from a file
  3. Background music from a CD

To select which option to use, click the appropriate box.  The area below the three options will change to show options relating to that setting.  These settings are explained below:
AUDIO CD

If you have an audio CD in your CD-ROM drive prior to selecting this option, you will be presented with the option of choosing to play the entire CD or only a specific track.  When playing a specific track, Picture Scroller will continuously repeat that track.  Otherwise it will repeat the entire CD.
MUSIC FILE

After selecting this option, to set the file you would like to use for the background music, you can either click the "Browse" button or type in the white box the specific path to the file.  Picture Scroller supports using either a wave file or a MIDI as the background music.  If you want to play an MP3 music file or any other format, run your music playing software in the background while scrolling through your pictures.  Also available when using a music file is the option to set whether to continuously play the music file or to play it only once.

Sound Effects

Located below the "Background Music" option, this button allows you to decide whether or not you would like to have Picture Scroller use its various sound effects throughout the program.
