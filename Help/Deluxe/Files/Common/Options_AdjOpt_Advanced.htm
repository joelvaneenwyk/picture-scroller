Adjust Options - Advanced

This section of options, available by clicking "Advanced" after selecting "Adjust Options", deals with various advanced options governing how the pictures are shown on the screen.

Screen Size & Color Depth

This option allows you to set the screen resolution that will be used while scrolling through pictures.  Each of the four possible choices are explained below:

PLEASE NOTE:  As these settings are advanced, it cannot be accurately determined whether a particular setting will work on your computer or not.  Until you are sure, it is advised that you leave the selection at "Use the Current Screen Settings".

"Use the Current Screen Settings"

 		Selecting this option, tells Picture Scroller not to adjust the screen settings in any way.  Whatever settings your computer is currently using will continue to be used throughout scrolling. (DEFAULT) 
"Optimize for Speed" Using this option, Picture Scroller will automatically select a setting that will make the transitions operate at their optimum speed.

NOTE:  For the sake of speed, picture quality may be drastically reduced as a result of using this option.
"Optimize for Quality" With this option, Picture Scroller will automatically select the best possible resolution for picture quality.

NOTE: For the sake of quality, transitions may run considerably slower using this option.
"Custom" Allows you to manually select the screen resolution that you would like to use.

Scrolling Colors

Here you can adjust the background and text color that will be used when scrolling through pictures.  To change the color of either one, click its color box.  When you are finished selecting the color, click "OK".
