Making a Picture Your Background (Wallpaper)

At any time while you're scrolling through pictures, you have the option to make the picture that is currently being shown become your background.  To do this, start by pressing the "F3" key on your keyboard.  A window will appear allowing you to choose to "Copy Picture to Windows Directory" or "Use Picture from Current Location".

Copy Picture to Windows Directory

This means that instead of directly making this picture your background, the picture is first copied into your Windows directory.  This way, if the picture is no longer available for some reason (such as, if the picture was on a CD), Windows can still continue to use the picture as your background.  However, if the picture is actually on your hard-drive, you probably don't want to use this option, because, in copying the picture, you would have two copies of the same picture on your computer.

New File Name

When using the above option, the file name that the copy will go by is shown here.  Typically, you don't need to change this; the default value should work just fine.

Use Picture From Current Location

This is just the opposite of the option above.  Using this, Picture Scroller will directly make the current picture your background.  You should probably use this option if the pictures you're scrolling through are already on your hard-drive.  One exception to this, however, is if the picture you've selected is not in bitmap format.  In this case, you must use the previous option, because Windows does not support any picture format except bitmap for use as wallpaper .

After you've selected from these options, click "OK".
