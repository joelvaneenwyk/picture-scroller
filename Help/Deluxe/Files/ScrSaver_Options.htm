Setting Screen Saver Options

To adjust the screen saver options, start by showing the "Display Properties" window as described in Enabling the Screen Saver.  Now, click "Settings", which is to the right of where you selected the screen saver.  Clicking that will bring up Picture Scroller in "Screen Saver Configuration Mode".  To select the pictures that you want to use for the screen saver, just add them as you normally would.  Picture Scroller will automatically save them into a list.  At this time, you can also set any other option available in Picture Scroller 2.1.  Note, however, that any option you select, aside from the pictures added, will apply both for the screen saver as well as for Picture Scroller in general.  When you're satisfied with the results, just exit Picture Scroller.  Your settings will be saved automatically.

Password Protection

Picture Scroller 2.1 supports using a password to protect your computer while you are away.  To enable password protection, see the Windows Help file.  You can do this by first clicking "Start"; then, clicking "Help".  Under the "Index" section, type "Screen Saver".  In the topics shown, you can find information about how to enable password protection.

For more information about the options available in Picture Scroller 2.1, see the section entitled Adjust Options.
