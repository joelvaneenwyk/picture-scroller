Select Pictures

This section explains the various options available when choosing your pictures.  To view these options, click the "Select Pictures" button on the left side of the main Picture Scroller screen.  At this point, the right side of the screen will change to show various buttons along the top with a blank area below.  Initially, this blank area will display a list of the folders of pictures as you select them.  If you want to see a list of the pictures inside a folder, double-click that folder.  To return to the list of folders, double-click the at the top of the list.

Add

The "Add" button is the first in the series along the top.  Clicking this button will reveal a list of options relating, quite obviously, to adding pictures.  The first option, "Directory", allows you to add an entire directory of pictures, including pictures in directories within the selected directory (sub-folders).  The second option, "File(s)", allows you to add individual pictures, one by one or as a group (Tip).

Picture Scroller supports many of the major picture formats, including bitmap (.bmp), JPEG (.jpg), GIF (.gif), PCX (.pcx), and Photoshop (.psd) pictures.  For a complete list of all the formats supported, click here.

The next two options, namely "Load Saved List" and "Save Current List", allow you to save in one file a list of all the pictures that you have added.  For example, let's say you have one hundred Yellowstone pictures that you've saved from different vacations over the past 5 years.  Instead of having to add all those pictures each time you want to scroll through them, you could add them once, save them as a separate list, and load them from the list each time you want to enjoy them.  To do this, just follow these steps:

  1. Add and organize all the pictures that you want using any of the methods given.
  2. Select "Add", and click "Save Current List".

    At this point, you are asked to specify where you would like to save the list.  After typing in a descriptive name, click "OK".

Now that you've saved the list, every time you want to scroll through them, do this:

  1. Select "Add", and click "Load Saved List".

  2. This time, just select the file you previously saved and click "OK".

Immediately, your pictures will be loaded into the list, ready for use!

Delete

The next button after "Add" is the "Delete" button.  Clicking this removes the selected picture(s) from the list.  If the list is showing folders (see above for explanation) and you select one of these, the entire folder of pictures will be removed from the list.  On the other hand, if you only want to remove individual pictures, first double-click on the folder, and then delete the specific pictures. (Tip)

Move Up/Move Down

These two buttons to the right of "Delete" allow you to adjust the scrolling order of the pictures.  Instead of simply having Picture Scroller set the order for you, you can use these buttons to move a picture up or down in the scrolling order, say, for example, into chronological order.  To do this, just select a picture (or folder) and click one of these buttons.

NOTE: You cannot use the SHIFT key or the CTRL key to move more than one picture at a time.

View

Clicking this button, allows you to quickly look at a picture.  After clicking "View", a white box appears just below the main buttons on the left side of the screen.  When you click on a picture from the list, this box changes to show a preview of the picture.
