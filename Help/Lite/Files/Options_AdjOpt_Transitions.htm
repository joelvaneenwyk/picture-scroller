Adjust Options - Transitions

After you have clicked the "Adjust Options" button, clicking "Transitions" will display the options available for choosing which transitions are used and in what order.  The white box on the left side lists all the featured transitions, while the box on the right will display the transitions that you have chosen.  To move the transitions back and forth between the two boxes, use the four buttons between them.  The top and bottom buttons move all the transitions in the designated direction, while the middle buttons them the individually selected transitions back and forth.

Above and below the box on the right, there are two arrows - one pointing up and another pointing down.  These allow you to adjust the order in which the transitions will be used.  Just select a transition from the right box and click either the up arrow or the down arrow to move it. 

Below these options, these is another option called "Transitions Appear Randomly".  If this option is checked , the transitions you have placed in the box to the right will be implemented in a random order.

Below is a list of all the 10 transitions available in this Lite version of Picture Scroller:

Blinds Vertical Diagonal Slide Right Up
